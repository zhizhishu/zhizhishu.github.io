Flux 3x-ui Orchestrator

一个基于 Flux Panel 的新主控项目：在一个工作台里选择多台服务器，自动安装/配置 3x-ui，把 VLESS Reality、VMess WS、Trojan TLS、Shadowsocks、Snell 和远端端口转发放进统一规则视图，并统一回看 Xray、证书、流量与服务状态。

查看仓库 一键安装

0.5.0 可公开上架，直接一条命令安装

主控优先拉取 GitHub Actions 发布到 GHCR 的镜像；如果 GHCR 暂未公开，安装脚本会回退到下载公开源码并本地构建镜像。默认端口是前端 80、后端/agent 回调 6365、phpMyAdmin 8066，均可在安装时改写。

主控面板

curl -fsSL https://raw.githubusercontent.com/zhizhishu/flux-3xui-orchestrator/main/scripts/install-master.sh | sudo bash

安装脚本会创建 `/opt/flux-3xui-orchestrator/.env`，下载 compose 和 `gost.sql`，然后拉取或构建镜像并启动 MySQL、后端和前端。

# Alpine / 极简系统没有 bash 时
curl -fsSL https://raw.githubusercontent.com/zhizhishu/flux-3xui-orchestrator/main/scripts/install-master-bootstrap.sh | sudo sh
# 显式运维动作
curl -fsSL https://raw.githubusercontent.com/zhizhishu/flux-3xui-orchestrator/main/scripts/install-master.sh | sudo bash -s -- upgrade
curl -fsSL https://raw.githubusercontent.com/zhizhishu/flux-3xui-orchestrator/main/scripts/install-master.sh | sudo bash -s -- backup
curl -fsSL https://raw.githubusercontent.com/zhizhishu/flux-3xui-orchestrator/main/scripts/install-master.sh | sudo bash -s -- restore --backup-file /opt/flux-3xui-orchestrator/backups/flux-master-backup-YYYYMMDD-HHMMSS.tar.gz
curl -fsSL https://raw.githubusercontent.com/zhizhishu/flux-3xui-orchestrator/main/scripts/install-master.sh | sudo bash -s -- uninstall --yes

`upgrade` 会先备份再刷新 compose/sql 并重启；`uninstall` 只移除容器，保留安装目录和 Docker volumes。

控制平面拆成清楚的四层

保持 KISS 和 YAGNI：不把 Snell 硬塞进 Xray，而是在主控做统一节点抽象，底层分别适配 3x-ui/Xray API 和 Snell agent/systemd/OpenRC。

Control Server

登记主控/副控服务器，保存 agent token、3x-ui endpoint、API token、证书状态、服务状态、流量和心跳时间。

Protocol Node

把 Xray inbound 和 Snell 服务记录到 `protocol_node`，同一张节点列表里管理创建、删除、重启、同步和运行状态。

3x-ui Proxy

通过 Bearer token 操作 `/panel/api/inbounds/*`，读取 Xray config/outbounds，并可请求远端 Xray 重启。

Server Forward

把被控机端口转发记录到 `server_forward_rule`，由 agent 创建 systemd/OpenRC + socat 服务，并在规则总览里和出入站一起查看。

Deploy Task

主控生成一键编排脚本；授权副控 agent 拉取、执行、回传日志和元数据，避免不可追踪的远程改动。

入站和出站都能从主控发起

入站走 3x-ui Bearer API，出站读取从运行配置提取，保存则使用登录态保护的 Xray 模板接口。
